About CSS

CSS fully known as Cascading Style Sheets is used to style and layout the pages. It can be used to style things like the fonts, color, size, and spacing of the contents and even keeps animations. Click here to Learn More

CSS has some building blocks which which includes CSS Selectors, Box Model, CSS values and Units, Cascades and Inheritance, Sizing and how to style Backgrounds and Borders.

CSS Selectors

CSS Selectors are used to select/find the HTML elements we want to style. Some of the popular/basic selectors we use are:

  • Element Selector: These selects HTML elements based on the name.

  • Id Selector: This is used to select the id attribute of an HTML Element to select a specific Element. Becasue of it's uniqueness, it can only be used where specified. The did element can override all other element when used.

  • CSS Class Selector: This is used to select HTML elements with a specific class attribute. To be able to select elements with a specific class, we use a period(.) character followed by the class name.

  • CSS Grouping Selector: This selector selects all the HTML elements with the same style formats. Grouping the selectors with the same style format helps minimize the codes and it is seperated with a comma(,).
